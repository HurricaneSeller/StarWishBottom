### 1. 用户部分

#### 1.1 注册账号

- method: POST
- URL `./users/reg`
- Header
  - Content-Type
    - application/x-www-form-urlencoded
- Param
  - email
    - 电子邮箱（唯一）
    - String，不超过20位
  - name
    - 用户名（可重复，最大长度8字符）
    - String，不超过八位
  - password
    - 密码
    - String，不超过十位
- 返回值
  - code
    - Number
    - 状态信息
    - 200 为成功，其余均为失败
  - data
    - msg
      - String
      - 如果注册成功，为一些乱七八糟的不需要管的东西
      - 如果注册失败，为失败原因
    - user_id
      - Int
      - 注册成功时返回，每个用户唯一标识符
- 样例输出

```
{
    "code": "200",
    "data": {
        "msg": "注册成功！",
        "user_id": 11
    }
}
```

#### 1.2 登录账号

- POST

- URL `./user/login`

- 请求头

  - Content-Type
    - application/x-www-form-urlencoded

- 请求体

  - user_id
    - 前面分配的user_id
  - password
    - 密码
    - String

- 返回值

  - code

    - Number
    - 状态信息
    - 200 为成功，其余均为失败

  - msg

    - name
      - 用户名
    - email
      - 电子邮箱
    - isAdmin
      - 是否为管理员（拥有删除所有心愿的权利）

    - 如果注册失败，为失败原因，为 String

- 样例输出

```
{
    "code": "200",
    "data": {
        "msg": "密码正确",
        "data": {
            "user_id": 2,
            "name": "福报修行者",
            "email": "234567@qq.com",
            "password": "23456"
        }
    }
}
```

    - 必填
  - content

- 返回值

    - 状态信息
